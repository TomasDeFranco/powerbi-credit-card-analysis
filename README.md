# 💳 Credit Card Transactions Analysis - Power BI

## 📌 Project Overview

This project analyzes credit card transactions in Argentina to understand customer behavior and spending patterns.

## 📊 Dashboard

📂 Dataset Description

* Transactions data
* Customer information
* Categories and locations
* Dates and amounts

## ⚙️ Data Preparation

* Data cleaning in Power Query
* Handling missing values
* Data transformation

## 📐 Data Modeling

* Relationships between tables
* Star schema approach

## 📈 Key Metrics (DAX)

* Total Spending
* Average Transaction
* Transactions per Customer

## 🔍 Insights

* High spending categories
* Customer segmentation patterns
* Geographic trends

## 💡 Business Insights

* Opportunities for targeted marketing
* Optimization of product offerings
* Customer behavior trends

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX
